# Pattern Generator & Number Analyzer 🔢⭐

A beginner-friendly Python project that combines **pattern generation**
and **number analysis** into a simple interactive program.

This project demonstrates fundamental programming concepts such as
**loops, conditional statements, user input handling, and menu-driven
programs**.

------------------------------------------------------------------------

## 📌 Project Overview

This program provides a simple command-line interface where users can
choose between multiple options:

1.  Pattern Generator
2.  Number Analyzer
3.  Exit Program

It is designed to help beginners practice Python logic building and
control flow.

------------------------------------------------------------------------

## ✨ Features

### ⭐ Pattern Generator

-   Generates star (\*) patterns based on the number of rows entered by
    the user.
-   Demonstrates nested loops and pattern logic.

### 🔍 Number Analyzer

-   Accepts a starting and ending range.
-   Identifies whether each number is Even or Odd.
-   Calculates the sum of all numbers within the range.

### 🖥️ Interactive Menu

-   User-friendly menu system
-   Handles invalid inputs
-   Allows repeated execution until the user exits
